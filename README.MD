# Model App Ecomerce

## Mode Development

- ## Requirements
    
    - `Make, Docker, docker-compose.`

- ## Start and Build Commands

    - $ `make build`
    - $ `make start`

- ## Start

    - $ `make start`

- ## Stop

    - $ `make stop`

- ## Restart

    - $ `make restart`

- ## Logs

    - $ `make logs`
